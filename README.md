# 💻 Projeto Computadores - Windows Forms (C#)

Este projeto consiste em uma aplicação desktop desenvolvida em C# utilizando Windows Forms, com o objetivo de simular a montagem e compra de computadores personalizados.

O sistema permite ao usuário selecionar diferentes configurações de hardware e software, como tipo de computador (PC ou Notebook), processador, memória, armazenamento (HD) e acessórios opcionais. A partir dessas escolhas, o sistema calcula automaticamente o valor total da configuração e exibe um resumo detalhado da compra.

## 🚀 Funcionalidades

* Seleção do tipo de computador (PC ou Notebook)
* Escolha de processador (i3 ou i5)
* Configuração de memória (2GB, 4GB ou 6GB)
* Seleção de armazenamento (500GB ou 1TB)
* Inclusão de acessórios (Sistema Operacional, Antivírus e Office)
* Exibição dinâmica de imagem conforme o tipo selecionado
* Cálculo automático do valor total
* Exibição de resumo da compra
* Botão para limpar/resetar todas as seleções

## 🧪 Objetivo do Projeto

Este projeto foi desenvolvido com foco no aprendizado de:

* Estruturas condicionais (if/else)
* Manipulação de controles do Windows Forms
* Eventos (Click, SelectedIndexChanged)
* Organização de interface gráfica
* Lógica de cálculo baseada em entradas do usuário

## 📚 Importância do Projeto

Este projeto possui um valor especial na minha trajetória, pois foi o **primeiro contato direto que tive com programação**, desenvolvido durante meu curso técnico em Desenvolvimento de Jogos Digitais pela ETEC em 2021.

Revisitar e recriar este projeto atualmente representa não apenas uma evolução técnica, mas também uma forma de consolidar conceitos fundamentais e aplicar boas práticas mais modernas, especialmente com foco em qualidade de software (QA).
